#### HistoryAddress
| Parameter | Popis |
| ----------- | ----------- |
| **Street** | ulica osoby |
| **StreetNumber** | popisné čislo budovy |
| **City** | mesto |
| **ZipCode** | PSČ |
| **District** | okres |
| **Region** | kraj |
| **Country** | štát |
| **ValidFrom** | dátum, od ktorého je detekovaná platná vo formáte `RRRR-MM-DDTHH:MM:SS` |
| **ValidTo** | dátum, do ktorého je detekovaná vo formáte `RRRR-MM-DDTHH:MM:SS` – môže byť prázdny |
